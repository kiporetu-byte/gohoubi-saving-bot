# ご褒美貯金bot💰✨

## 📌 概要

日々の小さな行動（運動した・スタバ我慢したなど）をトリガーに、自動で自分の口座間で振替して貯金できるLINE連携アプリです。

ユーザーがLINEでメッセージを送るだけで、自分のごほうび貯金用口座に一定額（500円）が貯金され、現在の残高が即時に返信されます。

「残高照会」メッセーだけを送っても、現在のごほうび貯金用口座の残高が確認できます。

---

## 🎯 背景・目的

- 習慣化を楽しく続けたい
- 行動と貯金を紐づけてモチベーションを上げたい

---

## 🚀 主な機能（MVP）

### ✅ 行動 → 貯金

LINEで以下のメッセージを送信：

- 「運動した」
- 「スタバ我慢した」

→ 500円を貯金

以下のように返信：

「500円貯金しました。
現在の残高は 〇〇円 です。」

### 💰 残高照会

LINEで以下のメッセージを送信：

- 「残高照会」

→今の貯金額の照会を返す
